The YUMA source

Dependencies

In case you want to hack on the source code of the YUMA Annotation Server, you may want to familiarize yourself with one or more of its library dependencies:

  • RESTeasy: JAX-RS implementation used for the API controllers
  • Hibernate: persistence framework for attaching relational databases
  • Jena Semantic Web Framework: used for RDF (de-)serialization
  • Rome: RSS feed generation
  • Wicket: Web framework used for the GUI

Getting and Building the Source Code

You can check out the source code from the official home of the YUMA Annotation Framework on github. The YUMA Annotation Server is built with ANT. Use the default 'create-war' target to build the .war file.
