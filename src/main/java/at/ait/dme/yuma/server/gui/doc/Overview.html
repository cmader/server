Overview

The YUMA Annotation Server is the storage back-end of the YUMA Universal Media Annotation Framework. It is based on a RESTful service architecture and can be deployed with various relational database management systems (such as PostgreSQL or MySQL) or a MongoDB NoSQL database. Furthermore, the YUMA annotation server provides various additional services:

  • annotation search, including support for the OpenSearch protocol
  • a RESTful JSON API
  • a Linked Data interface (RDF/XML and N3)
  • Multitude of RSS feeds on annotations, annotated objects and users public content: annotations

Features

  • REST interface

  • Search API based on OpenSearch protocol
  • RSS feeds
  • Backend: relational DBMS such as PostgreSQL or MySQL or the MongoDB NoSQL database

Annotation Data Model

The YUMA Annotation Server is based on a flexible internal multimedia annotation data model.
