YUMA Annotation Server Overview

The YUMA Annotation Server is the storage back-end of the YUMA Universal Media Annotation Framework. It can be deployed with a relational database such as PostgreSQL, or a MongoDB NoSQL database. The YUMA Annotation Server provides the following services:

  • JSON REST API
  • Search
  • OpenSearch API
  • Linked Data interface (RDF/XML)
  • RSS feeds - annotation feeds on objects and users, reply feeds on annotations
